\doxysubsection{Detailed Description}
Class to interface with the DRV8873 H-\/\+Bridge Motor Driver IC from Texas Instruments. This IC has several operating modes, but this class uses the PWM mode (which is the device default). Configuration and status monitoring are done via a SPI interface, but motor power and inversion status are set via PWM over two pins.
