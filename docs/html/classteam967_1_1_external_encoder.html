Detailed Description

Class that provides a way to talk to externally connected encoders (ones that are not integrated into a motor controller or other peripheral).

Member Enumeration Documentation

◆ EncoderType

enum team967::ExternalEncoder::EncoderType

Enum describing the different ways to measure the encoder pulses. Single only uses one of the two channels, half uses both channels but measures only one edge of the waveform, and full uses both channels and measures both the leading and trailing edges of the waveform.
