9This header provides a class that allows for a more object-oriented approach to using the PS3 library for the ESP32 developed by
10jvpernis, available here: https://github.com/jvpernis/esp32-ps3. It allows for access to all of the buttons, analog sticks, and the
11battery level of the controller.
12
13In the future this class may be replaced with a more generic one, with subclasses for different controller types if more controller
14support is added.
