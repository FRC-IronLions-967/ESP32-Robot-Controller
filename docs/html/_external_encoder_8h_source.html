9This header provides a class that uses the ESP32's pulse counting peripherals to interact with a quadrature encoder. It was based off
10of madhephaestus's ESP32 encoder library, which can be found here: https://github.com/madhephaestus/ESP32Encoder/
11
12Unlike madhephaestus's library, this one also provides support for velocity measurements, which could prove useful, and provides the
13ability to set the number of clicks per rotation to get the number of rotations that have passed.
14
