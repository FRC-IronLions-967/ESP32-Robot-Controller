9 This header provides a class that uses the ESP32's pulse counting peripherals to interact with a quadrature encoder. It was based off
10 of madhephaestus's ESP32 encoder library, which can be found here: https://github.com/madhephaestus/ESP32Encoder/
11 
12 Unlike madhephaestus's library, this one also provides support for velocity measurements, which could prove useful, and provides the
13 ability to set the number of clicks per rotation to get the number of rotations that have passed.
