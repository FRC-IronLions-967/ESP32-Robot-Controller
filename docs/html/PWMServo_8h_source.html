9This header provides a class that provides similar functionality to the MotorController class, but commands servos to absolute
10positions instead. Currently the class is rather limited, and only supports a range of motion of +-90° from the neutral position.
11This will likely be changed to allow for more flexibility. If you wish to run a servo in continuous mode, the PWM control is nearly
12identical to the PWM control for a motor controller, so you can just use the MotorController class to do so.
13
